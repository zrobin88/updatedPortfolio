  • The A-List

    The A-List is an application conceived from my experiences as a musician. The biggest issue facing bands and musicians is the sheer difficulty of finding the right people. The A-List aims to allievate that problem with a profile-based musician's network. The user will first create a profile, inputting the various required information (name, age, location, gender, instrument, musical style, etc.) and that profile will be stored in a mongo database. The user can then search for musicians based upon instrument, style, location and gender resulting in a list of matching profiles with contact information. The app utlizes mongoose, node and express to handle the data and back-end operations while the front-end UI utlizes react with a component-based architecture.

    Check out The A-List! Github Repo
  • CRUD Packer

    CRUD-Packer is a CRUD app for inventory and moving. This app is designed to help users keep track of box and storage unit contents. A user can create a collection and then add boxes to that collection. The user can input contents to the boxes and the application can look up box content based on an id system. The application utlizes node, mongo db, mongoose and express

  • LIRI Bot

    LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data. LIRI utlizes three API's (Spotify, Band is in Town and OMDB) in order to return data regarding music, concerts and movies. Be sure to read the instructions in the READ ME file in order to use the app properly.
