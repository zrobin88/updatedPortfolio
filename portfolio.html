My Very First App: "Concerto"

Concerto is an application for fans of music and concert-goers. Input the name of any band (or artist) and the app will return general information about the band/artist, the top ten tracks, two similar artists and upcoming live evetns with the option to purchase tickets.

LIRI Bot

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a _Language_ Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data. LIRI utlizes three API's (Spotify, Band is in Town and OMDB) in order to return data regarding music, concerts and movies. Be sure to read the instructions in the READ ME file in order to use the app properly.

GIF Madness

This application contains a series of "topic-buttons" that when clicked, populate the page with GIF's pertaining to the topic. This is done through GIPHY's API and Ajax calls. Als0, GIF's are set to animate when the user clicks on them. The app also allows the user to type in a topic and the app will dynamically generate a GIF button based upon the user's input. Go ahed and give it a try.

Train Scheduler

This application incorporates Firebase to host arrival and departure data. The app will retrieve and manipulate this information with Moment.js. This website will provide up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station.

Metal Band Name Generator

Being an avid heavy metal fan (and musician), this was a fun personal project to work on. The app is a simple name generator that pulls random indexes from two arrays for a two-word name and one index from a single arry for one-word-names. The muisc in the background (guitar, bass and drums) was performed and recorded by yours truly!
